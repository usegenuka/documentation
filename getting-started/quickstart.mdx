    <CodeGroup>

    ```bash Using npm

    ```

    Initialize the SDK in your app:

    ```javascript
    import Genuka from "genuka-api";
    
    const genuka = await Genuka.initialize({
      publicKey: "YOUR_PUBLIC_KEY",
      secretKey: "YOUR_SECRET_KEY",
      adminMode: true,
    });
    
    const orders = await genuka.orders.list({ isAdmin: true });
    console.log({ orders });
    ```

    2. **Using the REST API directly**

    You can also interact directly with the API using `fetch`, `axios`, or cURL:

    ```bash
    curl -X GET https://api.genuka.com/orders/ \
    -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
    ```
